## 🔧 How to Run

1. Clone this repository:
git clone https://github.com/EmilianoAyalaCanul/threejs-practice.git

2. Navigate to the exercise folder:
threejs-practice/01-cube

3. Install dependencies:
npm install

4. Run the project:
npm run dev
